# grpc-exception

gRPC exception in node.js

## Features

- Some Exceptions are already implemented:

  - Ok
  - Cancelled
  - Unknow
  - InvalidArgument
  - DeadlineExceeded
  - NotFound
  - AlreadyExists
  - PermissionDenied
  - ResourceExhausted
  - FailedPrecondition
  - Aborted
  - OutOfRange
  - Unimplemented
  - Internal
  - Unavailable
  - DataLoss
  - Unauthenticated
